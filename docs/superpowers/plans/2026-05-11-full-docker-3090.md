# Full Docker 3090 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Package the full ZeroGrasp + ManiSkill pipeline into a Docker image suitable for running on an RTX 3090 machine.

**Architecture:** Add a separate full Dockerfile so the existing ZeroGrasp-only Docker path stays intact. The full image uses one Python environment and runs `maniskill_codex.run_full_pipeline --no-conda`, with helper scripts for build/run and static tests that verify the expected Docker assets.

**Tech Stack:** Docker, CUDA 12.1 PyTorch base image, Python, ManiSkill/SAPIEN, ZeroGrasp CUDA extensions, shell scripts, unittest.
